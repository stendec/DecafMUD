{String} tr(decaf, obj, *args)
Translate a string from English to a different language, optionally replacing special character sequences with the provided variables.
Defined in: decafmud.js. 
alert( "This is a {0}.".tr(decaf, "test") );
// Example variable replacement using obj.
"{name} broke his {bone}!".tr({name: "Fred", bone: "tibia"});
// Becomes: "Fred broke his tibia!"
// Example variable replacement using multiple arguments.
"{0} broke his {1}!".tr("Fred", "tibia");
// Becomes: "Fred broke his tibia!"
Parameters:
{DecafMUD} decaf Optional
Use the language set in this instance of DecafMUD if set. If not set, use the language of the most recently created DecafMUD instance.
{Object} obj Optional
If an object is provided, variables will be replaced in the string based upon the object's keys.
{String|Number} *args Optional
If an object isn't provided, but additonal arguments are present, variables will be replaced in the string based upon the argument's index. Note that if the first argument is an Object, index-based replacements won't be made.
Returns:
{String} The translated text.
