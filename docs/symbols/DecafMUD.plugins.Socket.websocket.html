Class Detail
DecafMUD.plugins.Socket.websocket(decaf)

This provides support for HTML5 WebSockets to DecafMUD as a way to connect to servers. This, in theory, allows DecafMUD to connect without needing in-browser plugins such as Flash, Java, or Silverlight to supply a socket.

Unfortunately, WebSockets force us to use UTF-8 data, wrapped in a special protocol. And, while quite light, this extra protocol requires special software on the server to work.

See Green Lantern for letting WebSockets work with existing MUD servers.
