DecafMUD.plugins.Encoding.iso88591
This provides support for iso-8859-1 encoded data to DecafMUD, which isn't saying much as you realize that iso-8859-1 is simple, unencoded binary strings. We just have this so that the encoding system can work with a default encoder. 
alert(DecafMUD.plugins.Encoding.iso88591.decode("This is some text!"));
